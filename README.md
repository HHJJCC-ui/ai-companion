# 🤖 AI 智能伴侣

一个基于 **Streamlit** 和 **DeepSeek API** 构建的 AI 聊天应用，支持自定义伴侣角色、性格设定与会话历史持久化。

---

## ✨ 功能亮点

- 🎭 **自定义角色设定**：可自由配置伴侣昵称、性格风格，打造专属聊天体验
- 💬 **多会话管理**：支持创建/切换/删除会话，聊天记录本地持久化
- ⚡ **流式对话体验**：打字机式回复，模拟真实聊天交互
- 🔒 **隐私保护**：聊天记录仅保存在本地，不会上传到云端
- 🎨 **友好界面**：Streamlit 轻量化 Web 界面，上手零门槛

---

## 🚀 快速开始

### 1. 环境准备
确保你的电脑已安装 Python 3.8+，然后安装依赖：
```bash
pip install streamlit openai
2. 配置 API 密钥
前往 DeepSeek 开放平台 注册并获取 API Key
将密钥配置为系统环境变量：
Windows：set DEEPSEEK_API_KEY=你的API密钥
Mac/Linux：export DEEPSEEK_API_KEY=你的API密钥

3. 启动应用
streamlit run ai.py

📁 项目结构

├── ai.py              # 主程序文件（核心代码）
├── .gitignore         # Git 忽略配置文件
└── session/           # 自动生成的会话记录文件夹（不会提交到仓库）


📝 使用说明
新建会话：点击侧边栏「新建会话」按钮，创建全新聊天上下文
配置伴侣：在「伴侣信息」模块，修改昵称和性格，AI 会自动适配回复风格
切换会话：在「历史会话」列表，点击即可加载历史聊天记录
删除会话：点击会话右侧的删除按钮，即可移除无用会话

注意事项
请妥善保管你的 DeepSeek API Key，不要硬编码到代码中上传到 GitHub
聊天记录仅保存在本地 session 文件夹中，重装系统前请做好备份
如需部署到服务器，请确保配置了 HTTPS 环境，避免 API 密钥泄露
